The Doc Optimizer Server uses presigned S3 URLs for secure document uploads. This approach allows you to upload files directly to S3 without exposing AWS credentials.

## How it works

The upload process follows these steps:

<Steps>
  <Step title="Request a presigned URL">
    Call the `/api/upload-url` endpoint with your document's content type and desired optimization level.
  </Step>
  <Step title="Upload your file">
    Use the presigned URL to upload your document directly to S3 via a PUT request.
  </Step>
  <Step title="Start processing">
    Use the returned `jobId` and `key` to initiate document processing.
  </Step>
</Steps>

## Request a presigned URL

Send a POST request to `/api/upload-url` with the following parameters:

<ParamField path="contentType" type="string" required>
  The MIME type of your document (e.g., `application/pdf`)
</ParamField>

<ParamField path="optimizationLevel" type="string" required>
  The optimization level: `low`, `medium`, or `high`
</ParamField>

### Example request

```bash
curl -X POST https://your-api.com/api/upload-url \
  -H "Content-Type: application/json" \
  -H "x-api-key: your-api-key" \
  -d '{
    "contentType": "application/pdf",
    "optimizationLevel": "medium"
  }'
```

### Response

```json
{
  "uploadUrl": "https://s3.amazonaws.com/bucket/uploads/original/abc-123?signature=...",
  "key": "uploads/original/abc-123",
  "jobId": "abc-123"
}
```

<Note>
  The presigned URL expires after 60 seconds. Upload your file immediately after receiving the URL.
</Note>

## Upload your document

Use the `uploadUrl` from the previous response to upload your file:

```bash
curl -X PUT "https://s3.amazonaws.com/bucket/uploads/original/abc-123?signature=..." \
  -H "Content-Type: application/pdf" \
  --data-binary @document.pdf
```

<Warning>
  Make sure the `Content-Type` header matches the content type you specified when requesting the presigned URL.
</Warning>

## Using JavaScript

Here's a complete example using JavaScript:

```javascript
// Step 1: Request presigned URL
const response = await fetch('https://your-api.com/api/upload-url', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': 'your-api-key'
  },
  body: JSON.stringify({
    contentType: 'application/pdf',
    optimizationLevel: 'medium'
  })
});

const { uploadUrl, jobId, key } = await response.json();

// Step 2: Upload the file
const file = document.getElementById('fileInput').files[0];
await fetch(uploadUrl, {
  method: 'PUT',
  headers: {
    'Content-Type': 'application/pdf'
  },
  body: file
});

console.log('Upload complete! Job ID:', jobId);
```

## Using Python

```python
import requests

# Step 1: Request presigned URL
response = requests.post(
    'https://your-api.com/api/upload-url',
    headers={'x-api-key': 'your-api-key'},
    json={
        'contentType': 'application/pdf',
        'optimizationLevel': 'medium'
    }
)

data = response.json()
upload_url = data['uploadUrl']
job_id = data['jobId']

# Step 2: Upload the file
with open('document.pdf', 'rb') as f:
    requests.put(
        upload_url,
        headers={'Content-Type': 'application/pdf'},
        data=f
    )

print(f'Upload complete! Job ID: {job_id}')
```
